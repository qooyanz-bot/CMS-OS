# CMS-OS 営業先・提携候補台帳

調査日: 2026-07-13
対象: CMS-OS の企業Web、AIコンテンツ運用、業務用電話の販売提携先

## 目的

法人設立、バーチャルオフィス、会計・労務、創業支援の利用者に対して、CMS-OS の次のサービスを提案できる営業先を記録する。

- 企業Webサイトの初期構築
- CMS-OSによる記事、採用、PR、会社情報の運用
- SEO、AI検索向けコンテンツ、FAQ、構造化データの整備
- 代表電話、固定電話、電話転送、電話秘書と連携した業務用電話

この台帳の提携可能性は、公開情報から作成した営業仮説であり、各社との提携可否を確認済みという意味ではない。

## 提携条件の基本方針

### コミッション

- 基本案は、WEBと電話を皮切りにしたジェネリックSaaS群の自社サービス売上に対する50%コミッション。
- Web初期構築費、CMS利用料、AIコンテンツ運用費などを対象にする。
- ドメイン、サーバー、電話回線・通信料、キャリア原価、広告費、決済手数料、法定費用、税金、返金分はコミッション対象外とする。
- パートナーが販売、契約獲得、初回説明、一次問い合わせ対応まで担う場合は、継続課金への50%を基本案とする。
- 紹介だけの場合は、50%を永続化せず、初回契約または最初の12か月など別条件にする。

### 提携方式

| 方式 | パートナーの役割 | 50%の扱い |
|---|---|---|
| 紹介 | 顧客紹介、商談機会の提供 | 初回または最初の12か月を基本に協議 |
| 販売代理 | 提案、契約獲得、一次対応 | CMS-OS自社サービス売上の50%を基本案 |
| ホワイトラベル | パートナーブランドで販売、一次対応 | 価格・サポート範囲を個別設計 |

### 法務・表示上の注意

- 登記申請、税務、行政書士・司法書士業務の報酬に、顧客紹介の対価を混在させない。
- Web・電話の提案契約、請求、コミッションは、登記・税務・法務サービスと別契約・別項目にする。
- 電話サービスは、通信事業者、電話番号、転送、電話秘書、初期費用、通信料の責任分界を契約書で定める。
- 50%は提案条件であり、各社の既存代理店制度、原価、サポート負担、独占条件を確認してから確定する。

## 初回提案の型

### 提案する商品

- 新設法人向けCompany Web: 初期構築、会社情報、採用、問い合わせ、SEO基本設定
- CMS運用プラン: 記事、FAQ、採用、PR、更新代行、AI検索向け構造化
- 業務電話プラン: 代表電話、電話転送、電話秘書、問い合わせ導線
- バーチャルオフィス会員限定プラン: 住所利用開始から30日以内にWebと電話を立ち上げる

### 提携先に伝える価値

- 登記・住所契約の直後に発生する「会社の見た目」と「連絡先」の需要を取り込める。
- 住所・電話だけで終わらず、Web更新、採用、PR、問い合わせ対応の継続売上を作れる。
- パートナーは、CMS-OSの制作・運用基盤を利用し、自社で開発チームを持たずに商品化できる。
- 50%コミッションは、CMS-OSの自社サービス売上を対象にし、原価を除外して継続可能性を確保する。

## 営業管理項目

各社への接触後は、次の項目をこの台帳またはCRMに追加する。

| 項目 | 記録内容 |
|---|---|
| 接触日 | 初回メール、電話、フォーム送信日 |
| 窓口 | 会社名、部署、担当者、連絡先 |
| 顧客規模 | 会員数、拠点数、個人・法人比率など公開範囲で記録 |
| 現行商品 | Web、電話、会計、登記、電話秘書など |
| 競合状況 | 自社提供、外部提携、未提供 |
| 希望方式 | 紹介、販売代理、ホワイトラベル |
| コミッション | 50%の対象売上、期間、原価控除、支払条件 |
| サポート範囲 | 一次対応、制作、保守、解約、返金の担当 |
| 次回アクション | 商談日、提案資料、テスト導入、契約確認 |
| ステータス | 未接触、接触済、商談中、提案中、契約、見送り |
