# CMS-OS

AI 에이전트 네이티브 기업 콘텐츠 관리 플랫폼입니다.

[日本語](README.md) · [English](README.en.md) · [简体中文](README.zh-CN.md) · [Español](README.es.md) · [한국어](README.ko.md) · [Deutsch](README.de.md) · [Français](README.fr.md)

CMS-OS는 기업 정보, 채용, PR, IR, Blog를 하나의 플랫폼에서 관리합니다. AI 에이전트는 기획, 제안, 초안 작성, 사실 확인, 문장 다듬기, SEO 최적화, 정적 공개를 지원합니다.

CMS-OS는 현재 초기 개발 단계의 오픈 소스 프로젝트입니다.

## CMS-OS의 목표

CMS-OS는 검증된 기업 정보와 브랜드 가이드라인을 이해하는 AI 에이전트를 위해 설계되었습니다. 콘텐츠 목적, 독자, 산업, 지역, 포지션에 따라 검토와 공개가 가능한 콘텐츠를 제안하고 생성합니다.

## 주요 기능

- AI를 통한 주제, 콘텐츠 브리프, 구성, 초안, 최종 원고 생성
- 채용 포지션별 콘텐츠 생성
- PR, IR, Blog, 기업 정보, 미디어 자산 관리
- 승인된 기업 데이터와 출처를 기반으로 한 사실 확인
- SEO 제목, 설명, 내부 링크, FAQ, 구조화 데이터 생성
- 버전 이력, 승인, 감사 로그, 예약 공개
- 정적 HTML 생성 및 Cloudflare Pages 공개

AI 에이전트는 편집팀을 지원하며, IR, 법률 정보, 보수, 경영진 정보와 같은 민감한 콘텐츠에 대한 사람의 승인을 우회하지 않습니다.

## 산업 테마별 포털

CMS-OS는 산업 테마별로 사업자를 안내하고, 사용자, 발주자, 사업자, 리크루터 역할에 따라 표시 데이터와 작업 권한을 전환합니다. 현재 법률·변호사, 뷰티, 생성형 AI·업무 혁신, 인력 부족·인력 절감, 지역 관광·인바운드, 모빌리티 DX·SDV, GX·에너지 절감·자원 순환, 지역 활성화·이주·빈집 재생을 지원합니다.

- 사용자: 공개 사업자, 테마 가이드, FAQ 조회
- 발주자: 사업자 비교, 의뢰 작성, 견적 상담, 의뢰 이력 조회
- 사업자: 등록 정보, 채용, 문의, AI 콘텐츠, SEO, 공개 워크플로 관리
- 리크루터: 채용 공고와 사업자 조회, 지원, 지원 현황 확인

카테고리 목록과 확장 절차는 [카테고리 레지스트리](docs/CATEGORY-REGISTRY.md)에서 관리합니다.

## API/MCP 우선

CMS-OS의 모든 작업은 버전이 관리되는 API 또는 MCP를 통해 실행할 수 있어야 합니다. 관리 화면에서만 실행 가능한 업무 기능은 만들지 않습니다.

| 영역 | API 및 MCP 지원 범위 |
|---|---|
| 콘텐츠 | 생성, 조회, 수정, 삭제, 검색, 버전 관리, 번역, 보관 |
| AI 편집 | 제안, 초안, 다듬기, 사실 확인, 요약, 번역, SEO 감사 |
| 워크플로 | 검토, 승인, 반려, 예약 공개, 공개 취소 |
| 미디어 | 등록, 조회, 변환, 권리 메타데이터 관리 |
| SEO | 메타데이터, canonical, 구조화 데이터, sitemap, robots, 링크 감사 |
| 공개 | 빌드, 미리보기, 공개, 상태 확인, 롤백 |
| 운영 | 작업, 재시도, Webhook, 권한, 테넌트 설정, 감사 로그 |

API는 버전이 관리되는 REST/JSON과 OpenAPI를 기본으로 합니다. MCP 도구는 API와 동일한 도메인 서비스를 호출하며 비즈니스 로직을 중복 구현하지 않습니다. 관리 화면, AI 에이전트, CLI, BuilderOS Adapter는 모두 API/MCP 클라이언트로 취급합니다.

## Cloudflare Pages 정적 공개

CMS-OS는 승인된 콘텐츠에서 정적 HTML, CSS, JavaScript, 이미지, JSON-LD를 생성하고 BuilderOS Adapter를 통해 Cloudflare Pages에 공개합니다.

CMS API, 관리 화면, AI 처리는 공개 정적 사이트와 분리하여 빠른 전송, 강한 SEO, 높은 가용성, 낮은 운영 비용을 목표로 합니다.

## OSS 개발 방향

CMS-OS는 기업 콘텐츠를 만들고, 승인하고, 공개하고, 재사용하기 위한 협업형 오픈 소스 기반을 지향합니다.

검증 가능한 사실, 추적 가능한 AI 결과, 사람의 승인, 감사 가능성, SEO, 접근성, 정적 전송, 벤더 중립적인 확장성을 중요하게 생각합니다.
